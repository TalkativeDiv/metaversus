# Metaversus Design
> A landing page built with NextJS, TailwindCSS, and Framer Motion.

## Features

- 🎨 Simple and Beautiful UI using TailwindCSS.
- 🖌️ Simple yet Amazing transitions using Framer Motion
- 💾 Serverlessly Hosted on Vercel.
